
ベクトルはすべてワールド座標におけるものとする。

・拡散反射光(diffuse)
	物体内部に入り(屈折)、何度も散乱しながら物体表面に出てきた光。
	・強さ
			n	: 反射点(面)の法線ベクトル(正規化済み)
			v1	: 反射点から光源へのベクトル(正規化済み)
		とする。
		光源からの光が面に向かってまっすぐ当たるとき(光源へのベクトルと法線ベクトルが一致するとき)、
		拡散反射光は最も強くなる。
		よって、光源へのベクトルと法線ベクトルの内積[0, 1]を拡散反射光の強さとして扱うことができる。
		なお、内積値[-1, 0]のとき、光源が当たっている面は視点から見えないので、一律で強さ0とする。
		・式: float d1 = saturate(dot(n, v1))
	・物体の色
		物体内部に入った光の一部の色は吸収される。
			diffuse	: 白色の光RGB(1, 1, 1)が入ったときに表面に出てくる光RGB
		また、物体の色にテクスチャを使用できる。
			texColor	: そのピクセルにおけるテクスチャの色RGB
		物体の色はdiffuseとtexColorの乗算で求める。
		ここで物体、テクスチャの色のデフォルト値をともにRGB(1, 1, 1)にすると、
		diffuseのみ、texColorのみの描画ができる。
		・式: float3 d2 = d1 * diffuse.rgb * texColor.rgb

・鏡面反射光(specular)
	物体に入らず、法線を挟んで反対側に反射した光。
	・強さ
			n	: 反射点(面)の法線ベクトル(正規化済み)
			v1	: 反射点から光源へのベクトル(正規化済み)
			v2	: 反射点から視点へのベクトル(正規化済み)
		とする。
			ハーフベクトル h = (v1 + v2) / 2
		が法線と一致するとき、鏡面反射光は最も強くなる。
		よって、ハーフベクトルと法線ベクトルの内積[0, 1]を鏡面反射光の強さとして扱うことができる。
		なお、内積値[-1, 0]のとき、鏡面反射光の進む先に視点はないので、一律で強さ0とする。
		・式: float s1 = saturate(dot(h, n))
			saturate(x)は[0, 1]へのclamp関数(HLSL)。
			内積値は[-1, 1]なので、max(0, x)でも同じだが、saturate(x)の方が早いor同じな場合が多いらしい
	・物体の粗さ(乱反射)
		物体表面が粗いほど光は乱反射し、視点に届く反射光は弱くなる。
			specularity:	物体表面の粗さ
		とする。
		specularityが1のとき、最も滑らかであり、値が大きくなるほど粗くなる。
		x[0, 1]、n[0, inf]において、nが大きくなるほどx^nは小さくなることを利用する。
		・式: float s2 = pow(s1, specularity)
	・色(光沢)
		鏡面反射光は基本的に物体内へ入らずに反射するため、物体の色の影響を受けない。
		しかし、金属の場合には一部の光が物体内へ入り、光が着色される。
		そのため、鏡面反射光に色を設定すれば、金属の光沢を再現できる。
			specular: 物体のRGB(光沢)
		・式: float3 s3 = s2 * specular.rgb

・環境光(ambient)
	光源からの光は面に直接届くものもあれば、壁などを反射して間接的に届くものもある。
		ambient	: 環境光(間接光)の色RGB
		color			: 面の色RGB(diffuse.rgb * texColor.rgb)
	この間接的に入射する光による反射を再現するには、color色の面に対しambient色の光が入射することを
	考えればよい。
	・式: float3 a1 = color.rgb * ambient.rgb

・光の色
		reflection	: ある反射光の色RGB
		lightColor		: 光の色RGB
	反射光の計算はRGB(1, 1, 1)の光が入った時のものであり、
	それらにlightColorを乗算することで光の色の影響を与えることができる。
	環境光はambientが光の色を表しているため、乗算不要。
	・式: float3 result = reflection.rgb * lightColor.rgb

・アルファ値
	基本的に1(不透明)。texColor.aやdiffuse.aによって指定するか、オブジェクト単位のCBVで指定する。
	描画順を制御できないため、オブジェクトモデル内で完結するもの(肌の赤みや模様など)以外は
	texColor.aやdiffuse.aを1にすべき。
	オブジェクト単位でのアルファ値が1でないものは、不透明オブジェクト描画後にカメラから遠い順に描画する。
